# weatherMail
将某个城市的天气情况做成一封邮件发送至你关心的人的邮箱。

## 下载源码后直接打开run.py文件运行！

### run.py文件中需要的参数：

#### city = '北京' # 要查询的城市地址
#### msg_to = ['111111111@qq.com', 'xxxxxxxxx@163.com']  # 收件人邮箱，可以写多个。<br>
#### msg_from = 'xxxxxxxx@163.com'   # 发件人邮箱
#### password = 'xxxxxx'  # 发件人邮箱授权码

#### 将这四个参数改成自己的信息之后，直接运行即可。
